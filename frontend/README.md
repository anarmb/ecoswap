# EcoSwap Frontend 🌿

> **Dando una segunda vida a tus objetos favoritos. Únete a la revolución del consumo responsable.**

EcoSwap es una plataforma de **economía circular** diseñada para fomentar la sostenibilidad y la comunidad a través de la compraventa de artículos de segunda mano.

## Características

- Registro y login de usuarios con autenticación JWT
- Búsqueda de productos por categoría y título
- Impacto ambiental en la ficha de producto
- Subida de productos con imagen
- Panel de perfil con historial de compras y ventas
- Confirmación de transacciones por parte del comprador
- Panel de administración para gestión de usuarios, categorías y transacciones
- Rutas protegidas con `ProtectedRoute` y `AdminRoute`
- Diseño responsive para móvil y escritorio

## Instalación

```bash
git clone https://github.com/anarmb/ecoswap.git
cd frontend
npm install
```
Crear archivo .env
Arranca el servidor de desarrollo:

```bash
npm run dev
```
